# 图标文件

请将以下尺寸的图标文件放在此目录：

- `icon16.png` (16x16 像素)
- `icon48.png` (48x48 像素)  
- `icon128.png` (128x128 像素)

## 生成图标

有两种方式生成图标：

### 方式 1：使用图标生成工具（推荐）

1. 在浏览器中打开 `generate-icons.html`
2. 点击每个尺寸对应的"下载"按钮
3. 将下载的 PNG 文件保存到此目录

### 方式 2：使用自己的图标

如果你有自己的图标文件，只需确保：
- 文件名分别为 `icon16.png`、`icon48.png`、`icon128.png`
- 尺寸匹配（16x16、48x48、128x128）
- 格式为 PNG

## 临时解决方案

如果暂时没有图标文件，插件仍然可以正常工作，只是工具栏图标会显示为默认图标。
